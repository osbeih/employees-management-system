# Employees Management System

An Employees Management System built using PHP and MySQL, designed to manage employee data efficiently and track attendance through a simple and user-friendly interface.

## Features:

Add, edit, and delete employee records

Upload and store employee profile images

Track employee attendance

Generate attendance reports

Clean and simple UI

Full database integration using MySQL

## Technologies Used:

PHP

MySQL

HTML5

CSS3

## Project Structure:

index.php – Display employees list

attendance.php – Manage attendance

edit.php / delete.php – Update and remove records

report.php – Attendance reports

db.php – Database connection

## How to Run:

Download or clone the repository

Move the project to the htdocs folder

Create a MySQL database and update credentials in db.php

Start Apache & MySQL using XAMPP

Open the project in your browser

 This project demonstrates practical implementation of CRUD operations and database integration using PHP.
